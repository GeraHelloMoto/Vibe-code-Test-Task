# DECISIONS.md

1. **FastAPI** выбран из-за простоты, производительности и встроенной документации (OpenAPI). Позволяет быстро разработать REST API.
2. **SQLite** использована как легковесная встраиваемая БД, не требующая отдельного сервера, что упрощает запуск через Docker. Для небольших данных и тестового задания достаточно.
3. **Аутентификация упрощена** до выбора пользователя на странице, чтобы сосредоточиться на основной логике. В реальном проекте можно добавить JWT.
4. **Атомарное обновление статуса** для предотвращения гонок реализовано через UPDATE с условиями на статус и assignedTo. Это гарантирует, что только один запрос изменит запись, остальные получат 409 Conflict.
5. **Структура проекта** разделена на модели, схемы, роутеры и статические файлы. Это облегчает поддержку и расширение.
6. **Docker Compose** обеспечивает единое окружение и простой запуск одной командой.
7. **Автотесты** написаны с использованием TestClient FastAPI, проверяют создание заявки и сценарий гонки.
